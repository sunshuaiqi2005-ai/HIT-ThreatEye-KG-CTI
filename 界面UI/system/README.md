# 威胁情报分析系统

这是一个综合性的威胁情报分析系统，提供多种功能模块用于威胁情报的分析和处理。

## 功能模块

1. **威胁情报问答**
   - 基于垂直领域的问答系统
   - 支持自然语言查询
   - 提供准确的威胁情报解答

2. **知识图谱构建与分析**
   - 支持威胁情报知识图谱的构建
   - 提供图谱可视化功能
   - 支持关系分析和推理

3. **防御规则生成**
   - 基于威胁情报自动生成防御规则
   - 支持多种规则格式输出
   - 提供规则验证和优化功能

4. **数据处理**
   - 支持多种格式的威胁情报数据导入
   - 提供数据清洗和预处理功能
   - 支持数据分析和导出

## 安装说明

1. 确保已安装Python 3.8或更高版本
2. 安装依赖包：
   ```bash
   pip install -r requirements.txt
   ```

## 使用方法

1. 运行主程序：
   ```bash
   python main.py
   ```

2. 在主界面选择需要使用的功能模块
3. 按照各模块的提示进行操作

## 系统要求

- Python 3.8+
- 操作系统：Windows/Linux/MacOS
- 内存：建议8GB以上
- 存储空间：建议10GB以上可用空间
